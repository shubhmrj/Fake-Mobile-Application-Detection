TUANDROMD Dataset

Tezpur University Android Malware Dataset from UCI ML Repository. 4,465 samples with 241 static features extracted from real APKs.

  • 3,365 malicious samples (75%)
  • 1,100 benign samples (25%)
  • Binary feature vectors (permission/API present = 1)
  • IEEE 2020 publication by Borah & Bhattacharyya

XGBoost Classifier

Gradient boosting ensemble optimized for malware detection with class imbalance handling via SMOTE.

  • Algorithm: XGBoost 2.0 with 500 estimators
  • Learning Rate: 0.05 with early stopping
  • Features: Top 50 via Random Forest selection
  • Validation: 80/20 stratified split, 5-fold CV

Production Architecture

Containerized Flask app deployed on Hugging Face Spaces with models versioned on HF Hub.

  • API: Flask 3.0 with Gunicorn
  • Container: Docker (Python 3.10-slim)
  • Hosting: Hugging Face Spaces
  • Models: HF Hub auto-download at startup
